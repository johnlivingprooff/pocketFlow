# pocketFlow User Flow

**pocketFlow** helps you take charge and control of your money by keeping everything on your device, offline-first.

---

## üöÄ Getting Started

### First Launch
When you open pocketFlow for the first time:
1. **Onboarding Screen** displays a brief welcome message
2. You're guided to **Settings** to set your default currency (required)
3. Once configured, you're ready to start tracking your money

### Initial Setup
- Navigate to **Settings ‚Üí Currency** and select your preferred currency (USD, EUR, GBP, etc.)
- Optional: Enable biometric lock in **Settings ‚Üí Security** for added privacy
- Start by creating your first wallet

## üíº Core User Workflows

### 1. **Wallet Management** (Home ‚Üí Wallets)

#### Create a New Wallet
1. Tap **Wallets** in the bottom tab
2. Press the **+ Add** button
3. Enter wallet name (e.g., "Cash", "Checking", "Credit Card")
4. Choose currency (inherits default if same, or select another)
5. Set opening balance (current amount of money in this wallet)
6. Tap **Create**

**Use case:** Track multiple bank accounts, cash, credit cards, or savings in different currencies simultaneously.

#### Manage Wallets
- **View Balance:** Each wallet displays its current balance and currency
- **Reorder:** Drag wallets up/down to customize order
- **Edit:** Tap wallet ‚Üí Edit to change name or opening balance
- **Delete:** Remove a wallet (data archived in backups)

---

### 2. **Record Transactions** (Home ‚Üí Add Transaction)

#### Log an Expense
1. Tap the **+ Add** button on home screen
2. Select **Expense**
3. Enter amount and choose wallet to deduct from
4. Pick a category (Food, Transport, Utilities, etc.)
5. Add optional notes or receipt photo
6. Tap **Save**

**Use case:** Track daily spending‚Äîgroceries, gas, coffee, subscriptions.

#### Log Income
1. Tap the **+ Add** button
2. Select **Income**
3. Enter amount and choose wallet to credit
4. Pick category (Salary, Bonus, Refund, etc.)
5. Add notes if needed
6. Tap **Save**

**Use case:** Record salary deposits, freelance payments, or money received.

#### Transfer Between Wallets
1. Tap the **+ Add** button
2. Select **Transfer**
3. Choose source wallet (where money comes from)
4. Choose destination wallet (where money goes)
5. Enter amount
6. Tap **Save**

**Use case:** Move money between accounts without affecting income/expense analytics.

#### Attach Receipt Photos
- When adding an expense/income, tap **Add Receipt**
- Take a photo with camera or select from gallery
- Receipt is saved locally with transaction
- View receipt in transaction details anytime

**Use case:** Keep proof of purchases for reference, taxes, or claims.

---

### 3. **View & Analyze** (Home Screen & Analytics)

#### Dashboard (Home)
The home screen shows:
- **Net Balance:** Total across all wallets
- **Monthly Overview:** Income vs. Expenses chart for the month
- **Wallet Cards:** Quick view of each wallet's balance
- **Recent Transactions:** Latest 5-10 transactions with quick preview
- **Time Period Selector:** Switch between today, this week, this month, last month, or custom range

#### Analytics Tab
Deep-dive into your finances:
- **Monthly Trends:** Line chart showing income and expenses over months
- **Category Breakdown:** Pie chart showing which categories consume most money
- **Financial Health:** Quick snapshot of net worth, spending patterns
- **Insights:** AI suggestions on spending habits and opportunities to save

**Use case:** Understand where your money is going and identify spending patterns.

---

### 4. **Search & Filter Transactions** (Transactions ‚Üí History)

1. Tap **Transactions** in the bottom tab
2. View full transaction history (newest first)
3. Use search bar to find by:
   - Amount (e.g., "25.50")
   - Notes/Description
   - Wallet name
4. Filter by:
   - **Category** (click category name to drill down)
   - **Time Period** (today, week, month, custom)
5. Tap any transaction to view details or edit

**Use case:** Recall past spending, verify charges, or audit transactions.

---

### 5. **Manage Categories** (Settings ‚Üí Categories)

#### View Default Categories
- pocketFlow includes preset categories for common spending: Food, Transport, Utilities, Entertainment, Groceries, Health, etc.

#### Create Custom Category
1. Navigate to **Settings ‚Üí Categories**
2. Tap **+ Add Category**
3. Enter category name
4. Assign a color/icon
5. Tap **Create**

**Use case:** Tailor categories to match your spending habits (e.g., "Coffee Runs", "Gym Membership").

---

### 6. **Budget & Goals** (Budget Tab)

#### Set Budget Limits
1. Tap **Budget** in the bottom tab
2. Select category
3. Set monthly spending limit (e.g., "$500 for Food")
4. pocketFlow tracks against this limit

#### Monitor Progress
- See spending vs. budget for each category
- Get alerts when approaching or exceeding limits
- View breakdown by category

**Use case:** Stay disciplined and intentional about spending in specific areas.

---

### 7. **Backup & Data Management** (Settings ‚Üí Receipts & Security)

#### Automatic Backups
- pocketFlow automatically backs up data periodically
- All data stored locally (no cloud required)

#### Manual Backup
1. Go to **Settings ‚Üí Receipts**
2. Tap **Backup**
3. File saved to device (can export via file manager)

#### Restore from Backup
1. Go to **Settings ‚Üí Receipts**
2. Tap **Restore**
3. Select backup file to restore
4. Confirm to reload all data

**Use case:** Protect against data loss; migrate to a new device.

#### Manage Receipt Storage
- View total receipt photos and storage used
- Clear old receipts if needed (keeping transaction records)
- Receipts automatically organized by date

---

### 8. **Security** (Settings ‚Üí Security)

#### Enable Biometric Lock
1. Go to **Settings ‚Üí Security**
2. Toggle **Biometric Authentication**
3. Use fingerprint/face recognition to unlock app

#### Password Protection (if enabled)
- Set a PIN or pattern lock
- Authenticate each time you open the app

**Use case:** Keep sensitive financial data private on shared or public devices.

## üéØ Common Use Cases & Scenarios

### **Daily Spending Tracker**
1. Buy coffee ‚Üí Tap **+ Add** ‚Üí Expense ‚Üí Food category
2. Later, fill in receipt photo
3. Before bed, check **Home** dashboard to see today's spend
4. At month-end, analyze **Analytics** to review habits

### **Managing Multiple Accounts**
1. Create wallets: "Checking", "Savings", "Credit Card"
2. Each transaction picks the relevant wallet
3. Dashboard shows total across all wallets
4. Transfer between wallets (doesn't count as income/expense)

### **Tracking Bills & Subscriptions**
1. Set up recurring transactions (or manually add each month)
2. Use consistent category (e.g., "Subscriptions")
3. Check **Transactions** ‚Üí Filter by category to see all subscriptions
4. Use **Analytics** to spot trends and cancel unused services

### **Expense Reimbursement**
1. Log expense from your wallet (e.g., "Team lunch: $100")
2. Add note: "Reimbursement pending from @John"
3. When reimbursed, log income from that person's wallet or cash
4. Transaction history and receipt photos serve as proof

### **Travel Spending**
1. Create separate wallet for "Trip Budget" in local currency
2. Log all expenses with receipt photos
3. At end of trip, view total spending by category
4. Share receipt photos or transaction list with travel mates

### **Monthly Financial Review**
1. On 1st of month, open **Analytics**
2. Review last month's income vs. expenses
3. Check which categories exceeded budget
4. Drill down into top categories to identify opportunities
5. Adjust budget limits if needed

## ‚ùì FAQ

**Q: Can I use pocketFlow offline?**  
A: Yes! Everything works offline. Data syncs and updates are all local.

**Q: How do I export my data?**  
A: Use **Settings ‚Üí Receipts ‚Üí Backup** to export a JSON file with all transactions.

**Q: Can I delete a wallet?**  
A: Yes, but its history is kept in backups. Consider transferring its balance before deleting.

**Q: What currencies are supported?**  
A: All major currencies (USD, EUR, GBP, JPY, etc.). Set default in Settings.

**Q: How do transfers work in analytics?**  
A: Transfers don't count as income or expenses‚Äîthey're internal movements between your wallets.

**Q: Can I recover deleted transactions?**  
A: If you have a backup, you can restore it. Otherwise, deleted transactions are gone.
